LE JAVASCRIPT QUEZACO ?

Tu sais ce que c'est une page web ? C'est ce que tu vois sur ton écran quand tu vas sur Internet, par exemple pour chercher des informations, regarder des vidéos ou jouer à des jeux. Il y a des pages web de toutes les formes, avec des images, du texte, des boutons, des liens, etc. Mais comment ces pages web peuvent-elles faire des choses ? Comment peuvent-elles réagir quand tu cliques sur un bouton, quand tu entres ton nom, ou quand tu veux jouer à un jeu ? Il faut un langage qui permette de donner des instructions aux pages web, de les faire changer, de les rendre intelligentes. Ce langage, c'est le JavaScript. Le JavaScript est un langage de programmation, c'est-à-dire un langage qui permet de créer des programmes. Un programme, c'est une suite d'instructions que l'ordinateur doit exécuter. Par exemple, tu peux écrire un programme qui dit à l'ordinateur de dessiner un cercle, de changer sa couleur, ou de le faire bouger. Le JavaScript sert à écrire des programmes pour les pages web. Il permet de créer des éléments interactifs, qui réagissent aux actions de l'utilisateur. Par exemple, tu peux utiliser le JavaScript pour faire apparaître un message quand tu cliques sur un bouton, pour vérifier si le mot de passe que tu as entré est correct, ou pour animer un personnage dans un jeu. Le JavaScript s'utilise avec les autres langages que tu connais déjà : le HTML et le CSS. Le HTML sert à mettre du contenu sur la page web, le CSS sert à le mettre en forme, et le JavaScript sert à le rendre dynamique. Les trois langages travaillent ensemble pour créer des pages web modernes et attrayantes.

VOICI QUELQUES SCHEMAS QUI MONTRENT A QUOI RESSEMBLE DU CODE JAVASCRIPT

// affiche un message dans la console
console.log("bonjour !");
// création d'une variable a
var a = 3;
// affiche la variable dans la console
console.log(a);

Ce code définie une variable a dont la valeur est "Bonjour !" et affiche cette variale
sur l'écran, c'est à dire Bonjour !
