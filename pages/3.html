LE CSS QUEZACO ?

Tu te souviens du langage HTML, qui sert à créer des pages web avec des balises ? Il permet de décrire le contenu et la structure d'une page web, mais pas son apparence. Par exemple, il ne permet pas de choisir la couleur, la taille, la forme ou la position des éléments. Pour faire cela, il faut utiliser un autre langage, qui s'appelle CSS, qui veut dire Cascading Style Sheets, ou en français, feuilles de style en cascade. Le CSS sert à mettre en forme les éléments HTML en appliquant des styles. Le CSS fonctionne avec des règles, qui sont composées d'un sélecteur et d'une déclaration. Le sélecteur permet de choisir l'élément HTML sur lequel on veut appliquer le style, et la déclaration permet de définir le style à appliquer. Par exemple, si on veut que tous les titres de niveau 1 soient bleus, on écrit cette règle CSS : La déclaration contient une propriété et une valeur, séparées par deux points. La propriété indique le type de style à appliquer, et la valeur indique le choix du style. Par exemple, la propriété color permet de choisir la couleur du texte, et la valeur blue permet de choisir la couleur bleue. Il existe de nombreuses propriétés et valeurs en CSS, qui permettent de contrôler différents aspects de la mise en forme des éléments HTML. Par exemple, on peut utiliser la propriété font-size pour changer la taille du texte, la propriété background-color pour changer la couleur de fond, la propriété border pour ajouter une bordure, etc. Le CSS est un langage très puissant et créatif, qui permet de personnaliser l'apparence des pages web. Le CSS est un langage qui évolue constamment, et qui propose de nouvelles fonctionnalités pour rendre les pages web plus belles et plus interactives. Le CSS est un langage qui te permet d'exprimer ta créativité, de jouer avec les couleurs, les formes, les animations, et de créer des pages web uniques et originales. Tu peux commencer dès maintenant à ajouter du CSS à tes pages HTML, et tu verras la différence !

<html lang="fr">

<head>

<meta charset="UTF-8">

<title>Document</title>

<style>
body{background:#eee;}
</style>

</head>

<body>

</body>

</html>

Ce code changerait la couleur de fond de la page du navigateur.
